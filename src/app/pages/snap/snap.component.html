How It Works

Overview

Snap Nourish is a Snap Academy final project which goal was to create a new feature as if it were to be added towards the Snapchat app itself. Beforehand, different teams of six scholars were made throughout the academy where each team had to focus on a separate topic upon creating the new feature. Each team was composed of two scholars from each different Snap Academy which consisted of two engineers, two designers, and two storytellers. Together the whole project was group effort in creating Snap Nourish.


As far as how this feature would’ve worked is that a user would’ve had to open snapchat maps as soon as they open the application. Once there on the top right side of the screen below the map settings icon an option for Snap Nourish would've appeared for users to tap into. In return this would have shown across the map various grocery stores, food pantries, and community fridges for users to navigate different resources regarding these locations.

Resources

Upon clicking one of the many locations found on the map while using the Snap Nourish feature a user can immediately find different resources available to them provided by that specific location. Such common resources a user would typically be able to find are:

  • CalFresh application for those that are in need
  • Directions in how to get to that location
  • Recipes a user would be able to manage to follow along in order to cook healthy meals based on availability of items in that particular location

One important note to be aware of regarding this feature is that while everything was being developed during the course of this project is that only a few locations were added to show the prototype. In the case an API would’ve been used then it would’ve been possible to expand more resources throughout this feature.
